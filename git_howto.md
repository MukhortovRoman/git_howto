# Подсказка по GIT

*Инициализация репозитория*
```
git init
```
*Добавление файлов*
```
git add
```
*Внесение изменений сообщением*

```
commit -m "Message"
```
*Просмотр истории коммитов с изменениями*
```
git log
```
*Просмотр истории коммитов*

```
git log --online
```

*переключение между коммитами*
```
git checkout
```
